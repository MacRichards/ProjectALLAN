WordPress Theme Help

Please contact me if you have questions so that the theme can be improved. If this help page can be improved, let me know. Note: If you downloaded the WordPress Theme prior to April 26, 2014 and wish to upgrade, you must completely remove the old version. New updates will be indicated in the WordPress dashboard.

These instructions start at installing WordPress. You can dowload WordPress here.

  1. Install WordPress and the ScoutTroop theme
    1. Install WordPress (Theme has been tested on WordPress 3.9.)
    2. Login
    3. Go to Appearance | Themes
    4. Click "Add New"
    5. Click "Upload Theme"
    6. Browse to find the scouttroop zip file and click "Install Now"
    7. Click "Activate"
  2. Set Some Initial WordPress Settings
    1. Go to "Settings"
    2. General Settings
    3. Site Title should be your troop name
    4. Tagline should be your charter organization
    5. Set timezone to your local time
    6. Discussion Settings
    7. Uncheck "Allow people to post comments on new articles"
    8. Permalink Settings
    9. Post Name
  3. Prepare Home and Blog Pages (If you are not using 'post-name' as permalink setting, be sure the slug is appropriately named.)
    1. Delete (trash) the Sample Page, any sample posts, and any sample comments
    2. Add a New page called "Home Page"
    3. Be sure the slug or name that appears as a Permalink is "home-page" (e.g. Permalink: http://www.example.com/home-page/ ) Put nothing in the content area because it will not display.
    4. Add a new page called "News"
    5. Be sure the slug or name that appears as a Permalink is "news" (e.g. Permalink: http://www.example.com) Put nothing in the content area because it will not display.
  4. Set Up the Top Menu
    1. Got to "Appearance" | "Menus"
    2. Create a new menu named "Top Menu"
    3. Check "Top Menu" after Theme Locations and Save
    4. Check "News" and "Home Page" and click "Add to Menu"
    5. Save the Menu
    6. Consider changing the Navigation Label of "Home Page" to "Home"
    7. If you add pages to your site, remember to add them to the menu.
  5. Add Homepage Content
    1. Create (or rename) a post category "Home Page Articles" and change the slug to "home-page".
    2. Create four posts with the category "Home Page Articles". These will be the text on the front page.
    3. They appear in reverse chronological order, so adjust the publish date/time to move around on the page.
    4. Never put other posts in these categories.
  6. More Settings
    1. Go to Settings | Reading
    2. Change Front page displays to "A static page"
    3. and set Front page to "Home Page"
    4. and set Posts page to "News"
    5. Click "Save Changes"
  7. Changing Front Page Image
    1. Crop your image to 660px by 408px
    2. On WP dashboard menu go to Pages then All Pages.
    3. Edit the Home page
    4. Under "Featured Image" click "Set Featured Image"
    5. Upload or select the image you want to use.
    6. Click "Set Featured Image"
    7. Update the page
  8. Sidebar & Footer
    1. Add widgets to the sidebar and footer in the standard WordPress way.
  9. Some Suggested Plugins
    1. Contact Form 7 by Takayuki Miyoshi
    2. Really Simple CAPTCHA by Takayuki Miyoshi
    3. Google Calendar Events by Ross Hanney (A stylesheet is included in the theme for this.)
    4. Simple Lightbox by Archetyped
    5. bxSlider integration for WordPress by MarvinLabs
    6. WP Super Cache by Donncha O Caoimh (Can really speed up your WordPress site.)

The Scout Oath

All of scouting is designed to promote physical, mental, and spiritual excellence. Troop 90 lives this. Every one of our scouts is somebody important. He holds himself to these standards:

  • Trustworthy
  • Loyal
  • Helpful
  • Friendly
  • Courteous
  • Kind
  • Obedient
  • Cheerful
  • Thrifty
  • Brave
  • Clean
  • Reverent
